# MediaServer Monitoring Solution

Набор инструментов для построения систем мониторинга и видеонаблюдения через WebRTC. Включает в себя специализированный SDK и панель просмотра камер

## Состав репозитория

- **`packages/media-client`**: **Viewer SDK** — библиотека для интеграции функций просмотра видео-потоков в веб-приложения.
- **`frontend`**: **Monitoring Panel** — пример реализации интерфейса мониторинга (многопоточный просмотр, сетка камер, управление сессией).

## Быстрый старт

Убедитесь, что у вас установлен Node.js (v18+). Из корневой директории выполните:

```bash
npm start
```
*Эта команда автоматически установит зависимости, соберет SDK и запустит демо-панель.*

## Структура команд (Root)

| Команда | Описание |
| :--- | :--- |
| `npm start` | Полный цикл запуска (install -> build sdk -> dev frontend). |
| `npm run build:sdk` | Сборка SDK отдельно. |
| `npm run dev` | Запуск фронтенда (с авто-сборкой SDK при изменениях). |
