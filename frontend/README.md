# MediaServer Monitoring Panel (Viewer Demo)

Веб-приложение для мониторинга видео-потоков (камер) в режиме реального времени. Служит основным демонстрационным примером использования `@mediaserver/media-client` SDK.


## Быстрый старт (локально)

### 1. Подготовка SDK
Поскольку фронтенд использует локальную версию SDK, его необходимо собрать первым:

```bash
cd ../packages/media-client
npm install
npm run build
```

### 2. Запуск приложения
```bash
cd ../frontend
npm install
npm run dev
```

После запуска откройте `http://localhost:5173`. Вам потребуется токен доступа к узлу трансляции.

## Архитектура демо-приложения

- **`src/App.tsx`**: Основной компонент, управляющий подключением к сессии и жизненным циклом потоков через SDK.
- **`src/components/CameraView.tsx`**: Компонент для отображения отдельной камеры. Использует методы Subscriber из SDK для привязки видео.
